This combo allows you to bring up the Emacs frame from anywhere, even if the Emacs session is started from the terminal.

# Install

- Move `Bring up Emacs` app to `/Applications`
- Import the `Open "Bring up Emacs"` shortcut to Shortcuts app.
- Double click `Bring up Emacs` app to verify it works. It’ll prompt you for system event permission when running the first time, follow the instruction to enable accessibility permission for the app in System Settings.
- In Shortcuts, open the `Open "Bring up Emacs"` shortcut and click on the info button in the right panel, and change the shortcut labeled as `Run with`.
